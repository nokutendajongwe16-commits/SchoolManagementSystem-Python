# School Management System (Python)

This is a Python project that simulates a basic school management system. 
It allows managing students  by adding, editing , and viewing their information.

## Features
- Add new students 
- Edit students
- View all students that meet certain conditions
- Basic menu-driven interface

## How to Run
1. Make sure you have Python installed on your computer.
2. Open the folder and run the main file: `school_management.py`
3. Follow the on-screen menu to use the program.
